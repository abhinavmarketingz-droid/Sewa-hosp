# Operations & Scale Context

## Reliability
- Automated nightly backups and on-demand exports
- Health checks and uptime monitoring
- Rollback support for content and themes

## Performance Targets
- Lighthouse 99+ (Performance, Accessibility, SEO, Best Practices)
- Page loads under 1s (CDN + caching)
- Minimal JS for public pages

## PWA & Offline
- Installable PWA with offline cache
- Offline-first content cache strategy
- Push notifications (optional extension)

## Mobile Strategy
- React Native or Flutter shell
- Shared design tokens
- Offline content sync

## Multi-Tenant Ops
- Tenant quotas (media size, API rate limits)
- Usage dashboards
- Billing & invoicing integration

## Observability
- Structured logs
- Error tracking (Sentry/Datadog)
- Audit and security monitoring
